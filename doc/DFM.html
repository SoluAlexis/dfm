    class DFM

    Class instance of DFM generates a list of files recursively and indexes the by both MD5 hexdigest and file name Optional parameters include file extension filters with { :filters => [“jpg”, “png”] } and path suh as { :path => “./” }

    Attributes

    filters[RW]

    File extension filters can be set on DFM.new( { :filters => [“jpg”,“gif”] } ) or you should assign it to the class instance variable filters before calling recurse( path )

    recurse( path ) click to toggle source

    Start a fresh recursive search with empty hash indexes. Accepts parameter for path. (See filters for settings file extensions.)

    # File lib/dfm.rb, line 52
def recurse( path )
        @files_by_hexdigest = {}
        @files_by_name = {}
        recurse_path( path )
end
